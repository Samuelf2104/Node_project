# Node_project
Une fois le fichier téléchargé
Dans la console, aller sur l’emplacement du fichier.
Exécuter la commande: node server.js
La console renvoi une url que vous mettez sur votre navigateur et vous etes sur notre site internet !
La page d'accueil se se nomme index.html

Projet de Node React

Nous avons fait un site de vente de café et de thé en ligne. Nous avions commencé le projet avec la consigne du professeur de TD: « faites un site internet en vous aidant de ce qui a été appris en TD. » Nous avons donc poursuivi notez projet initial 
Le client peut se connecter et s’inscrire. Il peut ensuite choisir dans une sélection de café et de thé et ensuite passer commande. Quand une commande est passée, la commande est enregistrée dans une liste des commandes sous format json. De même tous les comptes clients sont enregistrés dans un json.

Nous avons fait un travail sur le css pour qu’il soit agréable et intuitif pour le client.
